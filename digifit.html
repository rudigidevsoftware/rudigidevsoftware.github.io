RudgiDevSoftware built the digifit application as an offline health, calorie, and macronutrient tracking application. This service is provided by RudgiDevSoftware and is intended for use as is.

This page is used to inform users regarding our policies with the collection, use, and disclosure of information if anyone decides to use our application.

Information Collection and Use

The digifit application does not collect personal user data.

All data entered into the application such as:

  • User physical profile (Name, age, height, weight)
  • Daily nutritional targets (Calories, water, protein, carbs, fat)
  • Daily food intake logs
  • Daily water consumption logs

is stored locally on the user's device and is not transmitted to any external server.

Device Permissions

The application may request certain permissions from the device, such as:

  • Storage access – used only to store application data locally on the user's device.

The application does not access personal files unrelated to the application.

Data Retention Policy

Since the digifit application operates completely offline and does not transmit data to external servers, we do not store or retain any of your personal data on remote servers. All data remains stored exclusively within the local storage of your device for as long as the application remains installed.

Data Deletion Process

Because your data is stored strictly locally on your device, you have full control over your data. You can delete your data at any time by utilizing any of the following methods:

  • Uninstalling the App: Uninstalling the digifit application from your device will automatically and permanently delete all locally stored data, profiles, and logs from your device.
  • Clearing App Data: You can manually clear all application data via your device's settings (Go to Settings > Apps > digifit > Storage > Clear Data/Clear Storage). This action will wipe all application logs and return the app to its default installation state.

If you have any questions or require assistance regarding data deletion, you may also submit a request via email to our support contact listed below.

Data Sharing

RudgiDevSoftware does not share, sell, or distribute user data to any third parties.

Because the application works offline, user data remains entirely on the user's device.

Data Security

We value your trust in using this application. All data is stored locally on the device. Users are responsible for maintaining the security of their device.

Children's Privacy

The digifit application is not specifically directed to children under the age of 13. The application does not knowingly collect personal information from children.
